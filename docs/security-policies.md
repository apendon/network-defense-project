## ── .✦ Network Security

- Firewalls restrict traffic between campuses
- Only approved connections are allowed between locations
- Dedicated router handles internet and cloud traffic
- Firewall placed between router and internet

---

## ── .✦ Access Control

- ACLs restrict access between VLANs
- HR and Legal databases are only accessible by authorized VLANs
- Access between international campuses is tightly controlled

---

## ── .✦ Encryption

- Data is encrypted in transit between campuses
- Sensitive data is encrypted at rest on database servers

---

## ── .✦ Identity & Authentication

- IAM implemented for access control
- Multi-Factor Authentication (MFA) required
- AAA (Authentication, Authorization, Accounting) enforced

---

## ── .✦ Monitoring & Detection

- Logs sent to SIEM for centralized monitoring
- EDR deployed on critical endpoints
- NDR monitors network traffic
- SIEM correlates data from multiple sources

---

## ── .✦ Device & Network Control

- NAC (Network Access Control) implemented
- BYOD devices are restricted and monitored
- Remote access secured via VPN

---

## ── .✦ Cloud Security

- Cloud access restricted via firewall rules
- Only permitted traffic allowed to/from cloud services


---
